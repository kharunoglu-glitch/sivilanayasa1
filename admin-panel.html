Admin / Supabase Kurulum

Supabase Entegrasyonu

Yorumları gerçekten kaydetmek ve admin panelini canlı veriye bağlamak için bu adımları izle

1
Supabase Hesabı Aç
supabase.com adresine git ve ücretsiz hesap oluştur. "New Project" ile yeni proje aç. Proje adı: sivilanayasa
Ücretsiz plan: 500MB depolama, 50.000 istek/gün. Yetecek — büyüyünce $25/ay Pro plana geç.
2
Veritabanı Tablolarını Oluştur
Supabase panelinde "SQL Editor"a git ve aşağıdaki kodu çalıştır:
-- Yorumlar tablosu CREATE TABLE yorumlar ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, madde_id TEXT NOT NULL, madde_no TEXT NOT NULL, madde_baslik TEXT NOT NULL, isim TEXT NOT NULL DEFAULT 'Anonim', sehir TEXT, oy TEXT CHECK (oy IN ('ol', 'olm', 'tar')), metin TEXT, durum TEXT DEFAULT 'bekliyor' CHECK (durum IN ('bekliyor', 'onaylandi', 'silindi')), ip_adresi TEXT, olusturma_tarihi TIMESTAMPTZ DEFAULT NOW() ); -- Güvenlik: sadece onaylananlar herkese görünür ALTER TABLE yorumlar ENABLE ROW LEVEL SECURITY; CREATE POLICY "Onaylananlar herkese açık" ON yorumlar FOR SELECT USING (durum = 'onaylandi'); CREATE POLICY "Herkes yorum ekleyebilir" ON yorumlar FOR INSERT WITH CHECK (true); -- Index: sorguları hızlandırır CREATE INDEX ON yorumlar (madde_id); CREATE INDEX ON yorumlar (durum); CREATE INDEX ON yorumlar (olusturma_tarihi DESC);
3
API Anahtarlarını Al
Supabase panelinde Settings → API sayfasına git. "Project URL" ve "anon public" anahtarını kopyala.
// sivilanayasa.html dosyasının <script> bölümüne ekle const SUPABASE_URL = 'https://PROJE_ID.supabase.co'; const SUPABASE_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6...'; // anon public key // Supabase client import { createClient } from 'https://esm.sh/@supabase/supabase-js@2' const supabase = createClient(SUPABASE_URL, SUPABASE_KEY);
4
Yorum Kaydetme Kodunu Entegre Et
sivilanayasa.html'deki gonder() fonksiyonunu şu şekilde güncelle:
async function gonder() { if (!secOy) { alert('Lütfen oyunu kullan!'); return; } const m = M[aktif]; const yorum = { madde_id: aktif, madde_no: m.no, madde_baslik: m.b, isim: document.getElementById('y-isim').value.trim() || 'Anonim', sehir: document.getElementById('y-sehir').value || 'Türkiye', oy: secOy, metin: document.getElementById('y-met').value.trim() }; const { error } = await supabase .from('yorumlar') .insert(yorum); if (error) { alert('Yorum kaydedilemedi. Lütfen tekrar deneyin.'); return; } // UI güncelle document.getElementById('y-met').value = ''; secOy = null; bildirimGoster(); }
5
Admin Panelini Canlı Veriye Bağla
admin-panel.html dosyasına aynı Supabase anahtarlarını ekle ve veriYukle() fonksiyonunu aktif et.
// admin-panel.html — gerçek veri yükleme async function gercekVeriYukle() { const { data, error } = await supabase .from('yorumlar') .select('*') .order('olusturma_tarihi', { ascending: false }); if (!error) { tumYorumlar = data; istatistikleriGuncelle(); yorumListesiniGoster(); } } // Onay/silme işlemleri async function yorumGuncelle(id, durum) { const { error } = await supabase .from('yorumlar') .update({ durum }) .eq('id', id); if (!error) gercekVeriYukle(); // listeyi yenile }
6
Vercel'e Yükle ve Domain Bağla
vercel.com'da ücretsiz hesap aç. HTML dosyalarını yükle. sivilanayasa.com domain'ini Vercel üzerinden bağla (domain kayıt: ~$10-15/yıl).
⚠️ admin-panel.html dosyasını herkese açık URL'de yayınlama! Ya ayrı bir subdomain'e koy (admin.sivilanayasa.com) ya da Vercel'in "Password Protection" özelliğini kullan.
